About Us

“ Debugging is twice as hard as writing the code in the first place. Therefore, if you write the code as cleverly as possible, you are, by definition, not smart enough to debug it.”

Programming today is a race between software engineers striving to build bigger and better idiot-proof programs, and the universe trying to build bigger and better idiots. So far, the universe is winning. Technology Mission at Gandhi Institute For Technology, Bhubaneswar, is a platform that provides group of programming enthusiasts and developers to work alongside their respective mentors in realtime projects.

Our programming developers are from various core branch of the Institute, who aim at proving the programming culture of the Institute. The students of the clubhave been proving their mettle in various fields pertaining to Computer Science.

  • We provide the right trick to solve the mistery behind any code.
  • We solve the challenges with a high demand to conquer
  • We set a bench mark towards the challenges and try to build the new era in the field of programming by creating new Tech aspirants.
